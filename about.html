About Movie Finding Made Easy

Movie Finder is your easy and fast way to discover, explore, and enjoy your favorite movies. This website was created as part of the Web Development 3.2 Final Project and demonstrates practical skills in HTML, CSS, JavaScript, and PHP.

Our Mission

We aim to make movie searching simple and fun by offering a clean, responsive interface where users can find information about movies, genres, ratings, and links to external resources like IMDb or YouTube trailers.

Technologies Used

  • HTML5 for structure and semantic layout
  • CSS3 for responsive and modern design
  • JavaScript for interactivity (search, filters, modals)
  • PHP for backend contact form processing

Design Philosophy

This site uses a dark, cinematic theme inspired by the atmosphere of movie theatres — combining clarity, contrast, and accessibility. Typography follows a clear hierarchy using <h1> through <h3> tags to meet web accessibility standards.
